# Overview
Measure the length of a string.    

# Install
Python 3 or higher is required.    
```cmd
# Linux/OS X
$ python -m pip install -U str-measure

# Windows
> py -3 -m pip install -U str-measure
```    

# Example
Measure the length.    
```py
import measure

print(measure.get_length("Hello, World!"))
```    
```
13
```
